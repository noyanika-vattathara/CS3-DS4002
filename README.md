# CS3-DS4002

#CS3-DS4002
This repository contains the study, deliverable, data, and reference materials necessary to complete the Case Study.

#Important Information
The two attached PDFs include the prompt for the case study as well as a rubric to follow in order to meet all requirements successfully.

#Data
The necessary data is located in the "DATA" folder and includes both a cleaned dataset to be used for your model as well as the original and US Energy Information Administration data.

#Supplemental Materials
The "Supplemental Materials" folder contains helpful resources related to the context of the issue and introductions to the VARMAX model and time series forecasting with XGBoost.

**#Scripts**
The "Scripts" folder contains 01_data_cleaning_py.py and 02_eda_starter_py.py. These are designed to help you begin the case study by preparing the dataset and initiating exploratory data analysis.
